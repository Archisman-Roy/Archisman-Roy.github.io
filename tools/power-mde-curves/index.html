Even Split divides traffic equally across all groups (control + treatments). Max Power allocates traffic to maximize power for control vs. any treatment. Control receives more traffic since it is shared across all comparisons. Optimal control share = 1 / (1 + √n), where n is the number of treatments (Dunnett allocation).
